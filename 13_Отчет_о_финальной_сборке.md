# Отчет о финальной сборке

1. Использованные файлы: вычитанный Markdown `07_Полный_черновик_ВКР_MindVault_вычитанный.md`, схемы из `pics_diploma/`, скриншоты из `vkr_assets/final_screenshots/`, код проекта MindVault, `docker-compose.yml`, package.json и TESTING.md.
2. Созданы скриншоты: fig07-fig12 в разрешении 1366x768, fig13 в мобильном viewport 390x845 при deviceScaleFactor 2, fig14 и fig15 в разрешении 1366x768.
3. Запущены проверки: `docker compose up -d --build`, `docker compose ps`, `corepack pnpm --filter @workspace/api-server test`, `corepack pnpm --filter @workspace/api-server run typecheck`, `corepack pnpm --filter @workspace/mindvault run typecheck`.
4. Итоговый документ содержит 83 страницы, 13 таблиц, 15 рисунков, 23 источника и 5 приложений.
5. Явных placeholders `[ВСТАВИТЬ...]` в DOCX не осталось. Даты выдачи задания и сдачи ВКР оставлены строками для ручного заполнения, так как точные даты не были предоставлены.
6. Перед печатью нужно вручную проверить оглавление, номера страниц и переносы широких таблиц.
7. DOCX собран как OOXML-пакет и открыт в Microsoft Word без ошибки.
8. PDF не создан: Microsoft Word COM открывал DOCX без ошибки, но экспорт `ExportAsFixedFormat` зависал дольше 180 секунд. Для PDF рекомендуется открыть готовый DOCX вручную в Word и выполнить "Сохранить как PDF".
